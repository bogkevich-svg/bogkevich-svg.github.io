# Відео — прев'ю зображення

Поклади сюди прев'ю (обкладинки) своїх відео.

- Формат: jpg, jpeg, png або webp
- Рекомендований розмір: 1280×720px або більше
- Називай файли: work-1.jpg, work-2.jpg, work-3.jpg...

Потім у index.html знайди масив `const works = [...]`
і встав шлях у поле `img`:

```js
{ title: 'Назва відео', cat: 'Video', large: true, img: 'images/video/work-1.jpg', accent: '#e63030' },
```

Щоб додати нову роботу — скопіюй один рядок і заміни дані.
