# Анімація — прев'ю зображення

Поклади сюди прев'ю (скріншоти або gif) своїх анімацій.

- Формат: jpg, jpeg, png, webp або gif (gif буде грати прямо в картці!)
- Рекомендований розмір: мінімум 800×600px
- Називай файли: anim-1.gif, anim-2.jpg...

Потім у index.html в масиві `const works`:

```js
{ title: 'Назва анімації', cat: 'Animation', large: false, img: 'images/animation/anim-1.gif', accent: '#e6c030' },
```
