# Логотипи — галерея

Поклади сюди зображення логотипів.

- Формат: jpg, jpeg, png або webp
- Рекомендований розмір: мінімум 800×800px
- Png з прозорістю — теж працює
- Називай файли: logo-1.jpg, logo-2.png...

Потім у index.html в масиві `const works`:

```js
{ title: 'Назва логотипу', cat: 'Logo', large: false, img: 'images/logo/logo-1.jpg', accent: '#30e660' },
```
